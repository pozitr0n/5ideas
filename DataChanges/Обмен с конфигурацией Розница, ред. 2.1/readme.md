### 1С:Предприятие 8
### Конфигурация "Управление небольшой фирмой"
#### Версия 1.5.2
=================================================================
### В этом файле содержится:

- Инструкция по подключению обновленных правил обмена между конфигурациями
"1С:Розница", редакция 2.1. и "Управление небольшой фирмой", редакция 1.5 

### Инструкция
-----------------------------------------------------------------

#### Правила конвертации.zip - комплект правил обмена, содержащий правила конвертации и регистрации объектов
для конфигурации "Розница".


Для загрузки в конфигурацию "Розница" правил конвертации объектов и правил
регистрации объектов необходимо:

1. Открыть меню "Администрирование". В панели навигации выбрать команду "Настройки синхронизации данных" и 
Перейти в форму управления обменами данных. 

2. Открыть обмен данными, для которого требуется изменить правила. 

3. На командной панели формы обмена данными, в зависимости от того, какие правила требуется изменить, 
выбрать команду "Загрузить правила".

4. В форме правил установить значение переключателя в положение "Из файла на компьютере"
и по кнопке "Загрузить" выбрать файл комплекта правил.

5. Обратить внимание, что загружены корректные правила, подходящие для используемых версий
конфигурации-источника и конфигурации-приемника.  

6. Нажать на кнопку "Записать и закрыть". Загруженные правила будут применены немедленно.

##### Примечание: комплект правил обмена используются для всех однотипных обменов данными (которые основываются 
на одном плане обмена; например: на базе плана обмена "Полный" настроено два обмена "Обмен с филиалом в Краснодаре" 
и "Обмен с представительством в Киеве"). Поэтому если в системе настроено несколько однотипных обменов, то замену правил
достаточно выполнить однократно (выбрав любой из настроенных обменов данными).
