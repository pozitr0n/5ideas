1С:Предприятие 8
Конфигурация "Управление небольшой фирмой"
Версия 1.5.2
=================================================================
В этом файле содержится:

- Инструкция по подключению обновленных правил обмена между конфигурациями
"Управление небольшой фирмой", редакция 1.5 и "Бухгалтерия предприятия", редакция 2.0.

Инструкция
-----------------------------------------------------------------

Правила конвертации.XML - Правила конвертации объектов для конфигурации "Бухгалтерия предприятия".

Правила регистрации.XML - Правила регистрации объектов для конфигурации "Бухгалтерия предприятия".

Загрузка указанных правил в конфигурацию "Бухгалтерия предприятия" может потребоваться в случаях,
описанных в файле совместного использования "Совместное использование.htm" в разделе "Контроль
актуальности правил обмена".

Для загрузки в конфигурацию "Бухгалтерия предприятия" правил конвертации объектов и правил
регистрации объектов необходимо:

1. Через меню "Сервис", подменю "Обмен данными с продуктами на платформе 1С:Предприятие 8.2",
подменю "Обмены данными" перейти в форму управления обменами данных. 

2. Открыть обмен данными, для которого требуется изменить правила. 

3. Из командной панели формы обмена данными в подменю "Параметры обмена данными", в зависимости
от того, какие правила требуется изменить, выбрать команду "Открыть правила конвертации объектов"
или "Открыть правила регистрации объектов".

4. В форме правил установить значение переключателя в положение "Правила из внешнего файла"
и по кнопке "Загрузить правила" выбрать файл правил.

5. Обратить внимание, что загружены корректные правила, подходящие для используемых версий
конфигурации-источника и конфигурации-приемника.  

6. Нажать на кнопку "Записать и закрыть".
Загруженные правила будут применены немедленно.

Примечание: правила конвертации объектов и правила регистрации объектов используются для всех
однотипных обменов данными (которые основываются на одном плане обмена; например: на базе плана
обмена "Полный" настроено два обмена "Обмен с филиалом в Краснодаре" и "Обмен с представительством
в Киеве"). Поэтому если в системе настроено несколько однотипных обменов, то замену правил
достаточно выполнить однократно (выбрав любой из настроенных обменов данными).

ВНИМАНИЕ! Пользователям, перешедшим с редакции УНФ 1.4 на редакцию УНФ 1.5, не требуется 
создавать дополнительных узлов обмена в информационной базе "Бухгалтерия предприятия 2.0". 
Достаточно в настроенных узлах обмена обновить правила конвертации и регистрации объектов.
